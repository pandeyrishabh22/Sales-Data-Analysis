Sales Data Analysis Project

Project Overview
This project analyzes a retail sales dataset to extract meaningful business insights such as revenue trends, regional performance, and top-selling products. The goal is to demonstrate data cleaning, exploratory data analysis (EDA), and data visualization using Python.

Tools and Technologies
Python  
Pandas  
Matplotlib  
CSV Dataset  

Dataset
The dataset contains retail transaction data including:
Order Date  
Product Name  
Category  
Region  
Sales  
Profit  

Business Questions Answered
What is the total revenue and total profit  
Which products generate the highest sales  
Which region performs best  
What are the monthly sales trends  
Which category contributes most to revenue  

Key Insights
Identified top 5 revenue-generating products  
Determined highest performing region  
Analyzed category-wise revenue distribution  
Visualized monthly sales trends  

How to Run This Project

1. Install Required Libraries
pip install -r requirements.txt

2. Place Dataset
Make sure sales.csv is in the same folder as sales_analysis.py

3. Run the Script
python sales_analysis.py

Graphs will open automatically.
